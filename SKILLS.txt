================================================================================
VS1024 スキルシステム v5.0 - 最終版
================================================================================

【基本仕様】
- スキル総数: 30種
- 装備上限: 5個まで
- コスト上限: 19-20
- 同じスキルは1個まで
- タイル生成時: 5%でスキル付与
- スキル付きタイル合成で確定発動
- 装備スキル内で比重に基づく抽選

【おすすめ構成例】
================================================================================

① バランス型（コスト20）
   ダブル(9) + ブースト(5) + スウィープ(3) + スクランブル(3)
   → Lv5が約60%で1回出る、安定構成

② ロマン型（コスト20）
   ダブル(9) + ガーディアン(9) + フリーズ(2)
   → Lv5が約94%で出る！超攻撃的

③ 安定型（コスト20）
   リフレクト(4) + ピュリファイ(4) + アンプリファイ(4) + スワップ(4) + ダブルエッジ(4)
   → 全スキル均等20%、ブレなし

④ 攻撃特化（コスト20）
   ダブル(9) + ウィークン(5) + アンプリファイ(4) + フリーズ(2)
   → 攻撃系メイン、Lv5が約59%で出る

⑤ 防御特化（コスト20）
   ガーディアン(9) + アーマー(5) + リフレクト(4) + フリーズ(2)
   → 防御系メイン、Lv5が約59%で出る

⑥ 効果重視（コスト19）
   ミラー(5) + ブースト(5) + スワップ(4) + ヴァニッシュ(3) + フリーズ(2)
   → Lv4が約19%、盤面操作特化

【スキル強化システム】
================================================================================

★0（初期）→ ★1: 同スキル1枚
★1 → ★2: 同スキル2枚
★2 → ★3: 同スキル2枚 + 同レア2枚
★3 → ★4: 同スキル3枚 + 同レア2枚
★4 → ★5: 同スキル3枚 + 同レア★3以上2枚

※強化による効果上昇は★5以外ゲームバランスを壊さない程度
