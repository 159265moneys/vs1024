================================================================================
                              通貨システム
================================================================================

【クリスタル】💎
- 用途: タイルガチャ
- 獲得方法: ステージクリア報酬

【SP（スキルポイント）】⚡
- 用途: スキルガチャ
- 獲得方法: マッチスコアに基づいて付与
- 計算式: 1マッチのスコアで約1回ガチャできる程度に補正
  → 例: スコア ÷ 300 = 獲得SP（300スコアで1SP、1回ガチャ=300SP）

================================================================================
                            スキル強化システム
================================================================================

【強化段階】
★0（初期）
  ↓ 同スキル1枚
★1
  ↓ 同スキル2枚
★2
  ↓ 同スキル2枚 + 同レアスキル2枚
★3
  ↓ 同スキル3枚 + 同レアスキル2枚
★4
  ↓ 同スキル3枚 + 同レア★3以上スキル2枚
★5（MAX）

【必要素材まとめ】
┌────────┬─────────────────────────────────────┐
│  段階  │           必要素材                 │
├────────┼─────────────────────────────────────┤
│ ★0→★1 │ 同スキル ×1                        │
│ ★1→★2 │ 同スキル ×2                        │
│ ★2→★3 │ 同スキル ×2 + 同レアスキル ×2      │
│ ★3→★4 │ 同スキル ×3 + 同レアスキル ×2      │
│ ★4→★5 │ 同スキル ×3 + 同レア★3以上 ×2     │
└────────┴─────────────────────────────────────┘

【★5までの総必要枚数】
- 同スキル: 1+2+2+3+3 = 11枚
- 同レアスキル: 2+2+2 = 6枚（★3以上含む）
